### Installazione

1. Clonare il repository nella destinazione desiderata

```shell
$ git clone https://github.com/ITISEnricoFermi/archivio-digitale-server.git
```

2. Installare le dipendenze

```shell
$ cd archivio-digitale-server/
$ npm install
```
